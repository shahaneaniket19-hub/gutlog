Bristol Stool Scale
Reference guide for stool types
⬛

Type 1 — Hard lumps

Separate hard lumps. Severely constipated. Increase water and fibre intake.

🌭

Type 2 — Lumpy sausage

Sausage-shaped with lumps. Mildly constipated. Slow transit time.

🥖

Type 3 — Cracked surface

Sausage with cracks. Normal range. Healthy for most people.

🍌

Type 4 — Smooth ✓ Ideal

Smooth, soft sausage. Gold standard. Indicates great gut health and hydration.

🫧

Type 5 — Soft blobs

Soft blobs, easy to pass. Borderline loose. Track trigger foods.

☁️

Type 6 — Mushy

Fluffy, mushy, ragged edges. Mild diarrhea. Common in IBS-D and gastrocolic reflex.

💧

Type 7 — Watery

Entirely liquid. Diarrhea. May indicate intolerance, infection or IBS flare. See a doctor if persistent.
